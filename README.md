# PickMe Driver App Test Automation

A scalable, maintainable Appium-based test automation repository for the PickMe Driver app.

---

## Project Structure

- `config/` — Application and environment configuration
- `core/` — Driver initialization, logging utilities
- `utils/` — Element locators and shared helpers
- `tests/` — Place your test scripts here
- `reports/` — Outputs and test reports

## Getting Started

1. Install dependencies:

    pip install -r requirements.txt

2. Set Appium and device config in `config/app_config.py`
3. Create your test cases in `tests/`.
